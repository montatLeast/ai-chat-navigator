# AI聊天我的发言导航器

Chrome 扩展：一键提取你在 Gemini / Grok / ChatGPT 里**自己**的所有发言，生成侧边栏列表，点击即可平滑滚动定位。

## ✨ 功能
- 自动识别并只显示**你的发言**
- 侧边栏列表，干净美观
- 点击任意一条 → 平滑滚动 + 高亮
- 新消息自动刷新，无需刷新页面
- 支持 Gemini、Grok、ChatGPT、Claude

## 📥 安装方式

### 方法1：从 GitHub Releases 下载（推荐）
1. 点击上方 **Releases** → 下载最新版本的 `ai-chat-navigator-v1.0.zip`
2. 解压到任意位置
3. 打开 Chrome → `chrome://extensions/`
4. 打开右上角「开发者模式」
5. 点击「加载已解压的扩展程序」→ 选择解压后的文件夹
6. 搞定！进入任意聊天页面，右上角会出现 📜 按钮

### 方法2：直接从源码安装（开发者模式）
同上，选中整个 `ai-chat-navigator` 文件夹即可。
